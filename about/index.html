About This Site

This is an unofficial site dedicated to Grid12, a new online multiplayer game by Jetbolt Games. The game is currently in closed beta; if you're interested in testing, please post here.

If you got here because you were redirected from the old Neocities address and you're looking for the Grid12 for Dummies guide, it's now under the wiki; click the link in the navigation header.

If you have any suggestions, send me a private message on the Grid12 forums, or fork the site repo and do a pull request!
